环境配置

镜像加速

  • 设置代理的方式
http_proxy=http://userName:password@proxyAddress:port

export http_proxy="http://127.0.0.1:1087"
export https_proxy="http://127.0.0.1:1087"
export all_proxy=socks5://127.0.0.1:1080
  • 设置国内源
cd "$(brew --repo)"
git clone git://mirrors.ustc.edu.cn/homebrew-core.git/ /usr/local/Homebrew/Library/Taps/homebrew/homebrew-core --depth=1
git remote set-url origin https://mirrors.ustc.edu.cn/brew.git
cd "$(brew --repo)/Library/Taps/homebrew/homebrew-core"
git remote set-url origin https://mirrors.ustc.edu.cn/homebrew-core.git
cd "$(brew --repo)"/Library/Taps/caskroom/homebrew-cask
git remote set-url origin https://mirrors.ustc.edu.cn/homebrew-cask.git
echo 'export HOMEBREW_BOTTLE_DOMAIN=https://mirrors.ustc.edu.cn/homebrew-bottles' >> ~/.bash_profile
source ~/.bash_profile
  • 恢复源
# 重置brew.git:
$ cd "$(brew --repo)"
$ git remote set-url origin https://github.com/Homebrew/brew.git

# 重置homebrew-core.git:
$ cd "$(brew --repo)/Library/Taps/homebrew/homebrew-core"
$ git remote set-url origin https://github.com/Homebrew/homebrew-core.git

安装node

brew update
brew doctor
brew link node
brew uninstall node
brew install node

安装淘宝代理

npm config rm proxy
npm config rm https-proxy
npm install -g cnpm --registry=https://registry.npm.taobao.org

全局安装

npm install -g hexo
npm install -g hexo-cli

项目配置

mkdir demo
cd demo
hexo init

本地运行

hexo s

部署到github

vi _config.yml
# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
  type: git
  repo: https://github.com/sulinyy/sulinyy.github.io
  branch: master

git插件

npm install hexo-deployer-git --save

推送git命令

hexo clean（清除缓存）
hexo g（上传到仓库）
hexo d（部署）

配置git

增加index.md，配置首页地址
index.html

git免密登录

ssh-keygen -t rsa -C 你的邮箱.com

https://github.com/settings/ssh/new
把id_rsa.pub代码复制到key上

创建新文章

// [layout] 为布局，可选项为 `post`、`page`、`draft`，这将决定文章所在文件路径。
// <title> 为文章标题
hexo new [layout] <title>

快捷命令

vi package.json
"dev": "hexo s",
"build": "hexo clean & hexo deploy",
"git": "hexo clean & hexo g & hexo d"

npm run build
npm run dev
npm run git

更换主题hexo-theme-matery

cd themes
git clone https://github.com/blinkfox/hexo-theme-matery
vi ../_config.yml
theme:hexo-theme-matery

配置类别

hexo new page "categories"
vi /source/categories/index.md
title: categories
date: 2020-10-14 12:00:00
type: "categories"
layout: "categories"

配置标签

hexo new page "tags"
